# vim-dirdiff

DirDiff plugin for Vim

If you like Vim's diff mode, you would love to do this recursively on two
directories!

## Installation

With [pathogen.vim](https://github.com/tpope/vim-pathogen):

    cd ~/.vim/bundle
    git clone git://github.com/will133/vim-dirdiff

## Usage

    :DirDiff <dir1> <dir2>
